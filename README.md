# Appointment App (React + Vite)

Minimal appointment booking app using React and Vite. Data persists to `localStorage` in the browser.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Open the URL displayed by Vite (usually http://localhost:5173).
